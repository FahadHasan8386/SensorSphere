📖 About

🌍 SensorSphere is a modern IoT monitoring platform built for learning and experimenting with real-world environmental monitoring solutions.

🔹 Monitor sensor data from DHT22, MQ7, and MQ136 sensors
🔹 Manage connected IoT devices from a centralized dashboard
🔹 Visualize temperature, humidity, CO, and H₂S trends through interactive charts
🔹 Explore authentication, role-based access, CRUD operations, and state management in Blazor WebAssembly
🔹 Designed with a scalable architecture that can later integrate ESP32, MQTT, ASP.NET Core API, SQL Server, and AI-powered analytics

⚡ Built with .NET 9, Blazor WebAssembly, Bootstrap 5, and C# while keeping the project beginner-friendly, clean, and ready for future SaaS expansion.
